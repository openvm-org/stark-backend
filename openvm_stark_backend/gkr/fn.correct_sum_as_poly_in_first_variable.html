Expand description

Computes r(t) = sum_x eq((t, x), y[-k:]) * p(t, x) from evaluations of f(t) = sum_x eq(({0}^(n - k), 0, x), y) * p(t, x).

Note claim must equal r(0) + r(1) and r must have degree <= 3.

For more context see Layer::into_multivariate_poly() docs. See also https://ia.cr/2024/108 (section 3.2).
